Detailed Description

The LSkatView class provides the view widget for the LSkatApp instance. The View instance inherits QWidget as a base class and represents the view object of a KTMainWindow. As LSkatView is part of the docuement-view model, it needs a reference to the document object connected with it by the LSkatApp class to manipulate and display the document structure provided by the LSkatDoc class.
