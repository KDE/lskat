Detailed Description

Monitor a port for incoming TCP/IP connections.

You can use a KRServerSocket to listen on a port for incoming connections. When a connection arrived in the port, a KSocket is created and the signal accepted is raised. Make sure you always connect to this signal. If you dont the ServerSocket will create new KSocket's and no one will delete them!

If socket() is -1 or less the socket was not created properly.
